# FirstCare Wildfire Support

FirstCare Wildfire Support is a web application built with React and Vite to help communities and responders access up‑to‑date wildfire information, resources, and support channels.

## Tech Stack
- React
- TypeScript
- Vite
- PostCSS / Tailwind

## Getting Started

### Prerequisites
- Node.js (LTS recommended)
- npm, pnpm, or yarn

### Install dependencies
```bash
npm install
```

### Run the development server
```bash
npm run dev
```
The app will typically be available at `http://localhost:3000/` (or as reported by Vite).

### Build for production
```bash
npm run build
```
The optimized production build will be output to the `dist` directory.

### Preview the production build
```bash
npm run preview
```

## Project Scripts
Common scripts defined in `package.json`:
- `npm run dev` – start the Vite dev server
- `npm run build` – create a production build
- `npm run preview` – preview the built app locally
- `npm run lint` – run linting (if configured)

## Development Notes
- Source code lives in the `src` directory.
- Static assets live in the `public` directory.
- Update this README as features are added (e.g. authentication, maps, alerts, contact forms).
